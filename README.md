# Power_BI_Dashboard

Interactive Power BI dashboard analyzing Ecom Express sales data to uncover revenue trends, product performance, and operational insights for data-driven decisions.

Project Overview
Developed using Power BI with SQL for ETL processes on 15K+ simulated orders across products, categories, states, and time periods. Key visualizations include KPIs (Total Revenue, AOV  12% cancellation rate), heatmaps, line charts for quarterly trends, and slicers for dynamic filtering.

Key Insights :
Top 20% products generated 40% revenue (Pareto principle), recommending inventory prioritization.
15% Q3 revenue dip tied to high cancellations (10% of total revenue lost), suggesting targeted promos for 25% uplift.
​Maharashtra contributed 25% revenue; electronics/laptops showed 18% YoY growth.
​

Tech Stack :
Tools: Power BI (Power Query, DAX), SQL, Excel

Skills Demonstrated: Data cleaning, KPI calculation, interactive visualizations, business storytelling
